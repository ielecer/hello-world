# Keyboard shortcuts for `latex + vscode` based on `macos` system

## Open keyboard shortcuts: `command+K command+S`
## Run pdflatex: `command+shift+B`
## Run bibtex: `command+shift+T`
## View latex pdf file: `control+option+V`
